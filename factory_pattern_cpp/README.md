
# Factory Method Design Pattern Example In C++

## 1. Build Setup 

### 1.0 CMAKE 
Make sure cmake is installed

### 1.1 Create build directory 
```
mkdir build 
```

### 1.2 Build Unit Test
```
cd .\build\ ; cmake .. ; cmake --build . ; cd ..

```
### 1.3 Run Unit Tests
```
 .\build\test_shape_factory.exe -v
```
